# KHAV 동물질병진단센터 — 프론트엔드 (1차)

건국대 가금질병 진단검사실(KHAV) 의뢰·결과 관리 시스템의 모바일 우선 웹.
**실 백엔드(Spring Boot) API 계약 기준**으로 개발했으며, 데이터는 현재 **교체형 mock 어댑터**(`src/adapters/mockApi.ts`)로 동작합니다(백엔드 배포 시 어댑터만 교체).

## 실행
```bash
npm install
npm run dev      # http://localhost:5173
```

## 데모 계정 (mock)
- 의뢰인: `vet` / `vet`  · 관리자: `admin` / `admin`
- 회원가입은 휴대폰 SMS 본인인증(mock — 화면에 인증번호 표시)

## 흐름
의뢰인: 가입(SMS 인증)·로그인 → 농장(도로명 검색) → 의뢰서(증상·검사·주령 자동) → 제출(검토중) → 진행/결과
관리자/검사자: 대시보드 → 접수확정(채번) → 결과등록 → 최종통보 → 통합검색 → CSV export

※ 체험용 mock(데이터는 각자 브라우저 저장). 실데이터 공유 운영은 백엔드 연동 후.
